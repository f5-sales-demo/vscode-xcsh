この拡張機能は、25のF5 XC APIドメインにわたる236以上のリソースタイプをサポートしています。リソースはツリービューでドメインカテゴリごとに整理されています。

## ドメインカテゴリ

- **アプリケーションデリバリー**: HTTP ロードバランサー、TCP ロードバランサー、オリジンプール、ヘルスチェック
- **セキュリティ**: Web アプリケーションファイアウォール、APIセキュリティ、ボット防御、DDoS保護
- **ネットワーキング**: 仮想ネットワーク、サイト、フリート管理、ネットワークポリシー
- **DNS**: DNSゾーン、DNSロードバランサー、DNSレコード
- **オブザーバビリティ**: ダッシュボード、アラート、ログレシーバー
- **インフラストラクチャ**: クラウドサイト、クラウドクレデンシャル、サービスポリシー

サポートされるリソースタイプの完全なリストは、F5 XC OpenAPI仕様から自動生成され、新しいAPIバージョンがリリースされると自動的に更新されます。
