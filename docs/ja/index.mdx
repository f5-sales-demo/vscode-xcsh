F5 Distributed Cloud Toolsは、エディターから直接F5 XCリソースの参照、作成、編集、削除を可能にするVS Code拡張機能です。25のAPIドメインにわたる236以上のリソースタイプに対して、完全なCRUD操作、スキーマバリデーション付きのJSON IntelliSense、オブザーバビリティツール、およびクラウドステータスモニタリングを提供します。
