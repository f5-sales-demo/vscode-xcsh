F5 Distributed Cloud Tools 是一款 VS Code 扩展，可让您直接在编辑器中浏览、创建、编辑和删除 F5 XC 资源。它提供完整的 CRUD 操作、带有 Schema 验证的 JSON IntelliSense、可观测性工具以及云状态监控，涵盖 25 个 API 域中超过 236 种资源类型。
