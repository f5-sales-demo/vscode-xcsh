## डिफ़ॉल्ट कीबाइंडिंग्स

कीबोर्ड शॉर्टकट्स को VS Code के Keyboard Shortcuts एडिटर (`Ctrl+K Ctrl+S` / `Cmd+K Cmd+S`) के माध्यम से कस्टमाइज़ किया जा सकता है। सभी एक्सटेंशन कमांड्स खोजने के लिए "f5xc" खोजें।

सभी 40+ कमांड्स Command Palette (`Ctrl+Shift+P` / `Cmd+Shift+P`) के माध्यम से "F5 XC" खोजकर एक्सेस किए जा सकते हैं।
