## Logs anzeigen

Klicken Sie mit der rechten Maustaste auf eine Ressource und wählen Sie **View Logs**, um Echtzeit-Logdaten von der F5 XC-Plattform zu streamen.

## Metriken anzeigen

Klicken Sie mit der rechten Maustaste auf eine Ressource und wählen Sie **View Metrics**, um Leistungs- und Datenverkehrsmetriken einzusehen.

## Architekturdiagramme

Wählen Sie **Diagram** bei einer Ressource, um ein visuelles Architekturdiagramm zu erstellen, das die Beziehungen und Abhängigkeiten der Ressource darstellt.
