F5 Distributed Cloud Tools ist eine VS Code-Erweiterung, die es Ihnen ermöglicht, F5 XC-Ressourcen direkt aus Ihrem Editor heraus zu durchsuchen, zu erstellen, zu bearbeiten und zu löschen. Sie bietet vollständige CRUD-Operationen, JSON IntelliSense mit Schema-Validierung, Observability-Tools und Cloud-Statusüberwachung für über 236 Ressourcentypen in 25 API-Domänen.
