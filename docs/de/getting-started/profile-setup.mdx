## Ein Profil hinzufügen

1. Öffnen Sie die Ansicht **Profiles** in der F5 XC-Seitenleiste.
2. Klicken Sie auf das **+**-Symbol, um ein neues Profil hinzuzufügen.
3. Geben Sie Ihre Tenant-URL ein (z. B. `https://your-tenant.console.ves.volterra.io`).
4. Wählen Sie eine Authentifizierungsmethode:
   - **API Token**: Fügen Sie Ihr API-Token aus der F5 XC-Konsole ein.
   - **P12 Certificate**: Wählen Sie Ihre `.p12`-Zertifikatsdatei aus und geben Sie das Passwort ein.
5. Legen Sie einen Standard-Namespace fest.

## Profile verwalten

- **Profile wechseln**: Klicken Sie auf ein Profil in der Profiles-Ansicht, um es zu aktivieren.
- **Bearbeiten**: Klicken Sie mit der rechten Maustaste auf ein Profil und wählen Sie **Edit Profile**.
- **Löschen**: Klicken Sie mit der rechten Maustaste auf ein Profil und wählen Sie **Delete Profile**.

Profile werden sicher über den integrierten Secret Storage von VS Code gespeichert (XDG-konform unter Linux).
