## การตรวจสอบสถานะ

มุมมองแบบต้นไม้ **Cloud Status** แสดงสถานะสุขภาพปัจจุบันของบริการ F5 Distributed Cloud สถานะจะถูกดึงข้อมูลจากหน้าสถานะอย่างเป็นทางการของ F5 และแสดง:

- สถานะรวมของแพลตฟอร์ม
- สุขภาพของแต่ละส่วนประกอบ (API, Console, Data Plane เป็นต้น)
- เหตุการณ์ที่กำลังดำเนินอยู่และช่วงเวลาบำรุงรักษา

มุมมองจะอัปเดตโดยอัตโนมัติตามช่วงเวลารีเฟรชที่กำหนดค่าไว้ (`f5xc.autoRefreshInterval`)
