## คีย์ไบนดิ้งเริ่มต้น

แป้นพิมพ์ลัดสามารถปรับแต่งได้ผ่านตัวแก้ไขแป้นพิมพ์ลัดของ VS Code (`Ctrl+K Ctrl+S` / `Cmd+K Cmd+S`) ค้นหา "f5xc" เพื่อดูคำสั่งทั้งหมดของส่วนขยาย

คำสั่งทั้งหมดมากกว่า 40 คำสั่งสามารถเข้าถึงได้ผ่าน Command Palette (`Ctrl+Shift+P` / `Cmd+Shift+P`) โดยค้นหา "F5 XC"
