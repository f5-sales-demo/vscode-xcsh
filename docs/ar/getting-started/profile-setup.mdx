## إضافة ملف تعريفي

1. افتح عرض **الملفات التعريفية** في الشريط الجانبي لـ F5 XC.
2. انقر على أيقونة **+** لإضافة ملف تعريفي جديد.
3. أدخل عنوان URL الخاص بالمستأجر (مثال: `https://your-tenant.console.ves.volterra.io`).
4. اختر طريقة المصادقة:
   - **رمز API**: الصق رمز API الخاص بك من وحدة تحكم F5 XC.
   - **شهادة P12**: حدد ملف الشهادة `.p12` الخاص بك وأدخل كلمة المرور.
5. قم بتعيين مساحة اسم افتراضية.

## إدارة الملفات التعريفية

- **التبديل بين الملفات التعريفية**: انقر على ملف تعريفي في عرض الملفات التعريفية لجعله نشطاً.
- **التعديل**: انقر بزر الماوس الأيمن على ملف تعريفي واختر **تعديل الملف التعريفي**.
- **الحذف**: انقر بزر الماوس الأيمن على ملف تعريفي واختر **حذف الملف التعريفي**.

يتم تخزين الملفات التعريفية بشكل آمن باستخدام التخزين السري المدمج في VS Code (متوافق مع XDG على نظام Linux).
