أدوات F5 Distributed Cloud هي إضافة لـ VS Code تتيح لك تصفح وإنشاء وتعديل وحذف موارد F5 XC مباشرة من المحرر الخاص بك. توفر عمليات CRUD كاملة، وإكمال ذكي لـ JSON مع التحقق من صحة المخطط، وأدوات المراقبة، ومتابعة حالة السحابة لأكثر من 236 نوع موارد عبر 25 نطاق API.
