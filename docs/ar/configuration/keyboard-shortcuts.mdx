## ارتباطات المفاتيح الافتراضية

يمكن تخصيص اختصارات لوحة المفاتيح من خلال محرر اختصارات لوحة المفاتيح في VS Code (`Ctrl+K Ctrl+S` / `Cmd+K Cmd+S`). ابحث عن "f5xc" للعثور على جميع أوامر الإضافة.

يمكن الوصول إلى جميع الأوامر التي يزيد عددها عن 40 أمرًا عبر لوحة الأوامر (`Ctrl+Shift+P` / `Cmd+Shift+P`) بالبحث عن "F5 XC".
