## Combinaciones de Teclas Predeterminadas

Los atajos de teclado se pueden personalizar a través del editor de Atajos de Teclado de VS Code (`Ctrl+K Ctrl+S` / `Cmd+K Cmd+S`). Busque "f5xc" para encontrar todos los comandos de la extensión.

Los más de 40 comandos son accesibles a través de la Paleta de Comandos (`Ctrl+Shift+P` / `Cmd+Shift+P`) buscando "F5 XC".
